PreSort Processing

Complete SOP for PreSort processing on JLS machines.

Overview

This document outlines the complete process for mail handling, including startup, operation, error handling, and shutdown of JLS postage systems. Follow each step carefully and always prioritize safety when operating the machine.

Warning: Always wear appropriate safety gear and do not attempt maintenance while the machine is powered unless instructed.

Startup Checks

  1. Verify the machine area is clear of obstacles and loose material.
  2. Ensure all safety guards are in place.
  3. Check that the machine is connected to power and the emergency stop is disengaged.
  4. Turn on the main machine power.
  5. Launch the JLS PreSort software on the connected computer.
  6. Verify the correct PreSort account and printer are selected.
  7. Run a test print of 1 item to confirm the printer is operational.
Warning: Do not start processing mail if test print fails or error lights are present.

Processing Workflow

  1. Load mail or material into the input hopper, ensuring proper orientation.
  2. Confirm that the mail is compatible with the selected postage type.
  3. Set the job parameters in the software (weight, postage class, quantity).
  4. Verify that the printer has enough ink and labels if applicable.
  5. Start the processing job using the software interface.
  6. Monitor the first few pieces to ensure proper printing and feeding.
  7. Continue processing in batches, checking periodically for jams or misprints.
Tip: Always process a small test batch after any configuration changes.

Troubleshooting & Common Errors

Excessive Postnet Print Errors

  • Error Code 0127: Too many postnet errors encountered. Check the printer head and connections.
  • Error .05695: Printer malfunction or paper jam. Inspect the printer and clear any jams.
  • Recommended Action:
    1. Check the Postnet printer for obstructions or dirt.
    2. Increase the allowed error rate in the software if appropriate.
    3. Perform a test print to confirm the issue is resolved.

Machine Jams

  • Stop the machine immediately.
  • Open the jam access points carefully and remove any stuck material.
  • Check sensors and rollers for obstructions.
  • Reset the machine before resuming processing.

Printer Errors / Ink Issues

  • Ensure sufficient ink or toner.
  • Check printer connectivity in the software.
  • Run a nozzle check or print test page.

Shutdown / End-of-Day

  1. Stop the current job and confirm all pieces are processed.
  2. Remove any remaining material from input/output hoppers.
  3. Turn off the JLS PreSort software.
  4. Power down the machine using approved shutdown procedure.
  5. Clean the machine area and remove any debris.

Notes & Tips

  • Run periodic test prints during long jobs to ensure consistency.
  • Always record error codes and resolutions for reference.
  • Keep consumables (ink, paper, labels) stocked to avoid interruptions.
